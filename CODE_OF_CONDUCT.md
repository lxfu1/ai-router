# 贡献者行为准则

## 我们的承诺

作为这个项目的贡献者和维护者，我们致力于为所有人提供一个友好、安全、包容的环境，无论其经验水平、性别、性别认同与表达、性取向、残障、外貌、体型、种族、民族、年龄、宗教或国籍如何。

## 我们的标准

有助于创造积极环境的行为包括：

- 使用友好和包容的语言
- 尊重不同的观点和经验
- 优雅地接受建设性的批评
- 关注对社区最有利的事情
- 对其他社区成员表示同理心

不可接受的行为包括：

- 使用带有性暗示的语言或图像，以及不受欢迎的性关注或性挑逗
- 挑衅、侮辱/贬低性评论，以及人身或政治攻击
- 公开或私下骚扰
- 未经明确许可发布他人的私人信息，如地址或电子邮箱
- 在专业环境中其他被认为不适当的行为

## 我们的责任

项目维护者有责任阐明可接受行为的标准，并对任何不可接受的行为采取适当和公平的纠正措施。

项目维护者有权利和责任删除、编辑或拒绝与本行为准则不一致的评论、提交、代码、wiki 编辑、问题和其他贡献，或暂时或永久地禁止任何贡献者从事其他他们认为不适当、威胁、冒犯或有害的行为。

## 适用范围

本行为准则适用于所有项目空间，也包括个人在公共空间代表项目或其社区时的情况。代表项目的例子包括使用官方项目电子邮件地址、通过官方社交媒体账号发帖，或在线上或线下活动中作为指定代表。项目维护者可以进一步定义和澄清项目的代表方式。

## 执行

如果你遇到或观察到滥用、骚扰或其他不可接受的行为，请通过 [GitHub Security Advisory](https://github.com/lxfu1/ai-router/security/advisories/new) 或发送邮件联系项目团队。

所有投诉都将被及时、公平地审查和调查。所有参与者都有义务保密与事件报告人相关的信息。具体执行政策的进一步细节可能会单独发布。

未真诚地遵守或执行行为准则的项目维护者可能会面临项目领导层其他成员所决定的临时或永久性后果。

有关本行为准则常见问题的答案，请参阅 https://www.contributor-covenant.org/faq。翻译版本可在 https://www.contributor-covenant.org/translations 查看。
